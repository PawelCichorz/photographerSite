# Photographer Site

A custom-built website for a photographer, created using **Next.js**. The project consists of both the frontend and API, with a focus on speed, interactivity, dynamism, and excellent responsiveness on mobile devices.

## Features

- **Speed and Optimization**: The site has been optimized for performance, ensuring fast loading times and smooth operation.
- **Interactivity**: The use of **framer-motion** provides animations that enhance the user experience.
- **Text Animations**: The **react-type-animation** library is used to dynamically display text with animated effects.
- **Responsiveness**: The site is fully responsive and works perfectly on mobile devices.
- **Smooth Scrolling and Links**: **react-scroll** is used to create smooth transitions between sections of the page.
- **Icons**: **react-icons/fa** has been used to add icons to the UI.

## Technologies

- **Next.js** – A React framework for building server-rendered applications.
- **Framer Motion** – Animations and interactions for React components.
- **React Type Animation** – For creating animated text effects.
- **React Scroll** – To enable smooth scrolling and navigation between sections.
- **React Icons** – A library for icons in React.
